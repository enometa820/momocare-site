02 / 콘텐츠 설계

매일 추천 조합 A1 + B1 + C1

가치 목적 타겟 효과
인지 변화 추적 보호자 단어 정답률 변화 → 의료적 시그널 감지
좋은 돌봄 행동 검증 보호자·시설 간병인이 친절하게 유도하는지 영상 증빙
영상 메시지 시니어 → 보호자 "감시"의 반대 — 시니어가 주도하는 콘텐츠
학대 방지 전체 매일 정해진 시간 녹화 = 학대 발생 시간대 특정 가능
03 / 촬영 체크리스트

구간별 필수 확인 항목

A
신체 상태 (1분) 점심 식사 직후 권장
  • 어르신이 스스로 보행하는 모습 5~10초 이상 촬영
  • 식사 중 반응 (손 떨림, 씹는 속도, 식욕) 촬영
  • 복약 후 30분 이내 반응 (표정·자세 변화) 기록
  • 카메라는 정면 또는 측면 45도, 얼굴+상체 프레임 유지
B
인지 활동 (2분) 오전 활동 시간 권장
  • 표준 단어 5개 불러주고 따라 말하기 (반복 여부 기록)
  • 오늘 날짜·요일 물어보기 (대답 여부 그대로 촬영)
  • 어르신이 기억하는 옛날 이야기 30초 이상 회상 유도
  • 간병인은 유도만, 대답을 대신하지 않기
매주 같은 5개 단어 사용 → 3개월 누적 시 인지 변화 추적 데이터 생성
C
대화 / 정서 (1분) 어르신 주도
  • "가족에게 하고 싶은 말 있으세요?" 질문 후 촬영
  • 오늘 기분 한 문장으로 표현하도록 유도
  • 억지로 말 시키지 않기 (무응답도 그대로 기록)
D
특이사항 (1분) 이상 없으면 환경 촬영
  • 낙상·통증 호소·이상 행동 있을 경우 즉시 촬영 후 보고
  • 특이사항 없으면 생활 공간 환경 상태 30초 촬영
  • 음성으로 "오늘 특이사항 없습니다" 한마디 녹음
04 / 촬영 원칙

공통 원칙

✔ 해야 할 것
  • ✓ 어르신 사전 동의 후 촬영
  • ✓ 자연광 또는 실내 조명 충분한 공간
  • ✓ 매일 같은 시간대 촬영 (루틴화)
  • ✓ 간병인 목소리는 명확하게 녹음
  • ✓ 배경 소음 최소화된 환경 선택
✖ 하면 안 되는 것
  • ✗ 어르신 의사 없이 강제 발화 유도
  • ✗ 인지 테스트 정답 미리 알려주기
  • ✗ 배경 소음이 심한 환경에서 촬영
  • ✗ 5분 초과해 촬영 (루틴 이탈)
  • ✗ 사후 편집·삭제 (원본 그대로 송출)
ℹ

이 매뉴얼은 단순 영상 저장이 아닌 의미 기반 이벤트 기록 체계입니다. 누적된 데이터는 AI 리포트 자동 생성, 노화 progression 추적, 학대 발생 시점 특정에 활용됩니다. 매일의 5분이 어르신의 건강 데이터가 됩니다.

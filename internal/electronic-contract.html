전자계약, 자체 개발이 답이다

한 줄 결론. Claude Code 페어 개발 + K-Startup 무료 법무 상담으로 외부 비용 사실상 0. 운영 시작부터 모두사인 대비 연 700만~7,000만 원 절감 (규모별).

왜 이 검토를 했나

7차 회의 modusign 도입 검토 액션

모두사인 건당 800원은 한국 시장 표준 단가다. 1팀이 보호자 약관·코디 계약·개인정보 동의 흐름을 본 도구로 구축하려고 검토 중인데, "800원이 비싸다"는 의문이 시작점이다.

모모케어가 1만 명 모객 시점에 도달하면 연 9만 건 이상이 발생한다. 본 비용은 결코 작지 않다.

자체 개발이 합법인가

2020-12-10 전자서명법 개정이 답을 열었다

2020년 12월 10일 전자서명법 개정으로 공인전자서명 제도가 폐지됐다. 그 전엔 공인인증서만 법적 추정력이 있었지만, 개정 후엔 모든 인증사업자가 발행하는 전자서명이 동등한 효력을 가진다.

일반 전자서명도 당사자 약정이 있으면 서명·기명날인으로서의 효력을 인정받는다. 자체 개발한 일반 전자서명도 모두사인과 같은 법적 효력을 가질 수 있다.

전제 조건. 분쟁 대비 감사 추적·해시 무결성 설계가 필수다. K-Startup 원스톱 지원센터 무료 법무 상담으로 충분히 검증 가능.

비용 — 외부 0원, 운영 변동비 55원

자체 개발의 본질은 비용 절감

초기 개발

· Claude Code 페어 개발 = 1팀 사용자 시간 1~2주

· 인프라 = Vercel·Supabase 무료 티어 (월 0원)

· 법무 자문 = K-Startup 원스톱 무료 상담 (백민혁 팀원 공유)

합계 사실상 0원 (1팀 시간 외 외부 비용 없음)

운영 변동비 (건당)

본인 인증 40원 + 카카오 알림톡 10원 + 인프라 5원 = 약 55원/건

모두사인 800원 - 자체 55원 = 건당 745원 절감

규모별 연 절감 (가정·1팀 검토 필요):
· 운영 초기 (각 1,000명, 연 9,500건) → 연 약 700만 원 절감
· 1만 명 모객 시점 (연 9.4만 건) → 연 약 7,000만 원 절감

결론

본 검토를 한 문장으로 줄이면

전자계약 시스템은 자체 개발이 답이다.

모두사인 800원은 한국 시장 표준이지만, 모모케어가 1만 명 모객 시점에 도달하면 연 7,500만 원 부담이 된다. Claude Code 페어 개발과 K-Startup 원스톱 무료 법무 상담을 활용하면 외부 비용 사실상 0으로 같은 기능을 구축할 수 있다. 운영 시작 시점부터 모두사인 대비 건당 745원이 절감된다.

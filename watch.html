MOMO CARE / 페어 워치 v0.2 — MVP 웰니스 안전권

의료기기 분류 회피형 최소기능(MVP) 설계

식약처 "의료기기 vs 웰니스 판단기준"에 따라 진단·측정 기능 제거. 호출·알림·통화 중심으로 재설계.

⚖️ 법적 포지셔닝 — 본 제품은 공산품(웰니스 기기)로 출시. 인증·신고 불필요. 표시·광고는 식약처 웰니스 가이드라인 준수.

3MVP 시나리오 (5개)

의료기기 분류 위험이 없는 시나리오만 유지. v0.1의 "S3 생체신호 이상"은 제거

응급 · 수동 S1. SOS 버튼 호출
대상자 워치 물리 SOS 버튼 3초 길게 누름 → 즉시 발신 + 자동 마이크 켜짐
간병인 워치 진동 + 알림음 + 양방향 음성 자동 연결 → BLE 신호 강도로 대략 위치
감지 · 알림 S2. 충격 감지 알림 (낙상 의심)
대상자 워치 6축 IMU가 강한 충격 + 무동작 5초 감지 → 대상자에게 "괜찮으세요?" 알림 → 10초 내 응답 없으면 간병인에게 알림
간병인 워치 "A실 김씨 충격 감지" 알림 → 음성 연결로 직접 확인
위치 S3. 연결 끊김 알림 (이탈 감지)
대상자 워치 BLE 신호 점차 약화 → 연결 끊김 → 마지막 신호 강도 기록
간병인 워치 "BLE 신호 약화 → 끊김" 단계별 알림. 마지막 연결 위치·시각 표시
알림 S4. 복약 / 일정 알림
대상자 워치 정해진 시각에 진동 + "약 드세요" 큰 글씨 → 확인 버튼 누르면 완료
간병인 워치 확인 시간 초과 시 알림 ("미확인 10분 경과")
업무 S5. 간병 업무 체크리스트
간병인 워치 오늘 할 일 (체위변경·식사보조·청결관리 등) 리스트 · 음성으로 완료 처리 ("완료" 한마디)
대상자 워치 해당 없음 (간병인 전용 기능)
표현 변경 핵심 — "낙상 감지" → "충격 감지", "이상 감지" → "알림", "측정" → "표시"로 모두 변경. 진단·판단을 사용자(간병인)에게 위임하는 구조.

5표시·광고 가이드 (필수)

웰니스 제품은 "의료기기 아님" 문구가 의무. 표현 하나로 의료기기로 재분류될 수 있음

❌ 쓰면 안 되는 표현
  • "낙상 감지" / "낙상 진단"
  • "심박 이상 측정" / "부정맥 감지"
  • "건강 모니터링" / "질병 예방"
  • "의료용" / "환자용"
  • "진단·치료·예방"
  • "생체신호 분석"
✅ 안전한 표현
  • "충격 감지 알림"
  • "심박수 참고 표시"
  • "안전 알림 기기"
  • "고령자용 / 일상 안전"
  • "호출·알림"
  • "활동·운동 참고"
필수 기재 사항 (식약처 웰니스 가이드라인) — 제품·앱·광고 모두에 다음 두 문구 명시 필요:
① "본 제품은 의료기기가 아닙니다"    ② "질병의 유무를 판단할 수 없습니다"

10참고 출처
